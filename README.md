# Image Morphing Tool using OpenCV

## 📌 Overview
This project implements an image morphing application using Delaunay triangulation to align facial landmarks and perform smooth image transformations. The system applies facial detection, geometric warping, and blending techniques to generate realistic morphing effects.

## 🚀 Features
- Face detection using OpenCV
- Delaunay triangulation for landmark mapping
- Image warping and blending
- Smooth transition between two facial images
- Output image generation

## 🛠 Technologies Used
- Python
- OpenCV
- NumPy
